3 Types of JavaScript Outputs

innerHTML

In the code, it says "innerHTML = 5 + 6;" meaning the javascript does the math part and changes the value.

document.write

document.write is another way to write down, in the code it says "document.write(5 + 6);" which is just telling the code to put down whatever is in the parenthesis.

Window Alert

The output will appear in a popup window ( shows when you first open the website ).
